# ⚽Football Project
<h3>계기</h3>
해외 축구 시즌 선수들이 다른 팀으로 이적하는 뉴스에 영감을 받아 선수 이적을 구현.
<h3>목표</h3>
선수의 소속팀을 변경하여 이적 구현하기
<h3>기능</h3>
1.회원가입, 로그인, 로그아웃
2.팀, 에이전트, 선수 등록
3.선수 목록, 팀 목록, 팀 별 선수 목록
4.선수 스카우트
5.선수 리뷰 작성, 리뷰 목록
6.유저 별 스카우트 목록
7.선수 계약<hr/>
